업체별 장단점

🏢 KG이니시스

  • 국내 최대 레퍼런스, 안정성 최상
  • 다양한 카드사·은행 연동
  • WPAY 간편결제 부가서비스
  • API 방식 빌링키는 별도 심사 (까다로움)
  • 초기 가입비 ~22만원
  • 문서가 복잡하고 오래된 편

🏢 NHN KCP

  • 정기구독 특화 레퍼런스 다수
  • 결제창 + API 방식 모두 지원
  • KCP PAY 간편결제 내장
  • 자동결제는 카드만 (계좌이체 불가)
  • 1년 후 수수료 3.2%로 오름
  • 별도 계약 필요

⭐ NICE페이먼츠

  • 카드 + 계좌이체(더빌CMS) 동시 지원
  • GitHub 공개 문서 + 샘플 코드 풍부
  • 해외 카드 결제 지원
  • 가상계좌도 지원
  • 초기 등록비 33~55만원 (비쌈)
  • API 직접 방식이라 개발 공수 필요

💛 카카오페이

  • 카카오톡 기반 높은 전환율
  • 0원 SID로 무료체험 구독 구현 쉬움
  • UX/UI 우수
  • 카카오페이 가입자만 결제 가능
  • 할부 불가
  • 수수료 비공개, 검수 절차 있음

💚 네이버페이

  • PG 중간 수수료 없음
  • 네이버 검색/쇼핑 생태계 노출 효과
  • 일반 + 정기결제 통합 지원
  • 네이버페이 사용자만 결제 가능
  • 검수 기간 평균 15영업일
  • 주문형/결제형 구분 필요 (복잡)

🔵 페이코

  • NHN KCP와 결합 운영 가능
  • 고도몰 등 NHN커머스 연동 용이

  • 포트원 미들웨어 의존도 높음

🔷 스마트로

  • 빌링키 단독 발급은 별도 계약
